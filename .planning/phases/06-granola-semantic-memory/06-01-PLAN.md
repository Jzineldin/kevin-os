<objective>
Implement the Granola transcript watcher (CAP-08 + AUTO-05). Every 15 min, poll Notion Transkripten DB with a `last_edited_time` filter; for each new page, read its content, publish a `transcript.available` event to `kos.capture`, and advance the cursor. Idempotency on `transcript_id` (Notion page id) prevents double-processing on EventBridge retries or operator manual triggers.

Purpose: feed Wave 2's transcript-extractor with a steady stream of new meeting transcripts, with provenance + idempotency that the rest of Phase 6 can rely on.

Output: working granola-poller Lambda + CDK helper that wires it into IntegrationsStack. The Lambda is fully testable with mocked Notion + mocked EventBridge.

    ```ts
    import { initSentry, wrapHandler } from '../../_shared/sentry.js';
    import { setupOtelTracingAsync, flush as langfuseFlush, tagTraceWithCaptureId } from '../../_shared/tracing.js';
    import { Client as NotionClient } from '@notionhq/client';
    import { ulid } from 'ulid';
    import { TranscriptAvailableSchema } from '@kos/contracts';
    import { getTranskriptenDbId, queryTranskriptenSince, readPageContent } from './notion.js';
    import { getCursor, advanceCursor } from './cursor.js';
    import { getPool, findPriorOkRun, insertAgentRun, updateAgentRun, publishTranscriptAvailable } from './persist.js';

    if (!process.env.AWS_REGION) process.env.AWS_REGION = 'eu-north-1';

    export const handler = wrapHandler(async (_event: unknown) => {
      await initSentry();
      await setupOtelTracingAsync();
      const ownerId = process.env.KEVIN_OWNER_ID;
      if (!ownerId) throw new Error('KEVIN_OWNER_ID not set');

      const notion = new NotionClient({ auth: process.env.NOTION_TOKEN! });
      const pool = await getPool();

      const dbId = await getTranskriptenDbId();
      const cursor = await getCursor(pool, ownerId);
      const since = cursor.lastCursorAt;

      let processed = 0;
      let skipped = 0;
      let maxLastEdited = since;
      const errors: string[] = [];

      try {
        for await (const page of queryTranskriptenSince(notion, dbId, since)) {
          const transcriptId = page.id;
          tagTraceWithCaptureId(transcriptId);

          if (await findPriorOkRun(transcriptId, 'granola-poller', ownerId)) {
            skipped++;
            continue;
          }
          const runId = await insertAgentRun({ ownerId, captureId: transcriptId, agentName: 'granola-poller', status: 'started' });
          try {
            const content = await readPageContent(notion, transcriptId);
            const detail = TranscriptAvailableSchema.parse({
              capture_id: transcriptId,           // page id used as capture_id (not ULID; allowed by schema)
              transcript_id: transcriptId,
              title: content.title,
              transcript_text: content.transcript_text,
              recorded_at: content.recorded_at.toISOString(),
              attendees: content.attendees,
              notion_url: content.notion_url,
              detected_at: new Date().toISOString(),
            });
            await publishTranscriptAvailable(detail);
            await updateAgentRun(runId, { status: 'ok', outputJson: { title: content.title, len: content.transcript_text.length } });
            processed++;
            const pageEdited = new Date(page.last_edited_time);
            if (pageEdited > maxLastEdited) maxLastEdited = pageEdited;
          } catch (err) {
            await updateAgentRun(runId, { status: 'error', errorMessage: String(err) });
            errors.push(`${transcriptId}: ${String(err)}`);
          }
        }

        if (processed > 0) {
          const cursorAdvance = new Date(maxLastEdited.getTime() - 60_000); // -1 min per RESEARCH §6
          await advanceCursor(pool, ownerId, cursorAdvance);
        }
      } finally {
        await langfuseFlush();
      }

      return { processed, skipped, errors: errors.length, since: since.toISOString(), advancedTo: processed > 0 ? new Date(maxLastEdited.getTime() - 60_000).toISOString() : null };
    });
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Notion API → Lambda | Notion responses are external input; Zod parsing on transcript_text length cap (64 KB) blocks oversized payloads |
| Lambda → kos.capture bus | PutEvents detail is Zod-validated before send |
| Scheduler → Lambda | EventBridge Scheduler trust boundary; scheduler role can only invoke this Lambda + the notion-indexer set per least-privilege grants |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-POLLER-01 | Tampering | Notion-controlled transcript_text | mitigate | Zod parse caps length to 64 KB; truncation in `readPageContent` before parse |
| T-06-POLLER-02 | Repudiation | Lambda runs without trace | mitigate | tagTraceWithCaptureId(transcript_id) on every page; agent_runs row per transcript |
| T-06-POLLER-03 | Denial of Service | Backlog burst on first run | mitigate | First-run cursor = now() - 24h cap; subsequent runs page through 100 transcripts/poll; next 15-min poll continues backlog |
| T-06-POLLER-04 | Elevation of privilege | Scheduler role misuse | accept | grantInvoke restricts to GranolaPoller Lambda only; trust scheduler.amazonaws.com matches Phase 1 pattern |
| T-06-POLLER-05 | Information disclosure | NOTION_TOKEN in env | mitigate | Token resolved from Secrets Manager via NOTION_TOKEN_SECRET_ARN; no plaintext env beyond Lambda runtime |
