<task type="auto">
  <name>Task 1: Generic retire-vps-script.sh tool</name>
  <files>scripts/retire-vps-script.sh</files>
  <action>
    Bash script, `set -euo pipefail`, ~80 lines:

    **Usage:** `bash scripts/retire-vps-script.sh --unit <name> --replaced-by <replacement> [--dry-run] [--ssh kevin@98.91.6.66]`

    **Flow:**
    1. Validate args (--unit is required, --replaced-by is required, --ssh defaults to kevin@98.91.6.66).
    2. **Dry-run path:** print the commands it WOULD execute over SSH; exit 0.
    3. **Real path:**
       a. SSH + `systemctl is-active <unit>` — if 'inactive' already, warn and skip stop.
       b. SSH + `sudo systemctl stop <unit>` (timeout 30s).
       c. Verify: `systemctl is-active <unit>` == 'inactive'.
       d. SSH + `sudo systemctl disable <unit>`.
       e. SSH + `sudo systemctl mask <unit>` (defensive — even manual start refuses).
       f. SSH + `journalctl -u <unit> -n 50` — capture last 50 lines, save to `.planning/phases/10-migration-decommission/retirement-logs/<unit>-<timestamp>.log`.
       g. **Write event_log audit row** via `psql $RDS_URL -c "INSERT INTO event_log(owner_id, kind, details, actor) VALUES ('kevin', 'vps-service-stopped', '{\"unit\":\"$unit\",\"replaced_by\":\"$replaced_by\",\"host\":\"98.91.6.66\"}', 'retire-vps-script.sh')"`.
       h. Print `[OK] retired $unit — replaced by $replaced_by`.

    **Safety:** If `psql $RDS_URL` fails, EXIT 1 BEFORE step b (audit-first per D-12). No systemd mutation without audit trail.

    **Env required:** `RDS_URL` (or read from `kos/rds-admin-url` secret via `aws secretsmanager get-secret-value` + jq), `SSH_KEY_PATH` (defaults to `~/.ssh/id_ed25519`).

    Include an `--undo` flag: `sudo systemctl unmask <unit> && sudo systemctl enable <unit> && sudo systemctl start <unit>` + event_log kind='vps-service-restored'.
  </action>

    <automated>bash -n scripts/retire-vps-script.sh && grep -q 'systemctl stop' scripts/retire-vps-script.sh && grep -q 'systemctl disable' scripts/retire-vps-script.sh && grep -q 'systemctl mask' scripts/retire-vps-script.sh && grep -q 'event_log' scripts/retire-vps-script.sh && grep -q -- '--dry-run' scripts/retire-vps-script.sh && grep -q -- '--undo' scripts/retire-vps-script.sh</automated>
