    **`src/archive.ts`** (~120 lines):
    - Export `canonicalizeWorkflow(w: object): string` — sorts keys recursively; stringifies with 2-space indent for human readability AND deterministic hashing (alternative: RFC 8785 JCS if a dep is allowed; else roll manual recursive sort).
    - Export `archiveWorkflow(deps: { s3, kms, pg }, w: { id, name, active, nodes, ... }, opts: { bucket, prefix, kmsKeyId }): Promise<ArchiveResult>`.
    - Flow:
      1. `const canonical = canonicalizeWorkflow(w);`
      2. `const sha = crypto.createHash('sha256').update(canonical).digest('hex').slice(0, 16);`
      3. `const key = `${prefix}/${w.id}-${sha}.json`;`
      4. `await s3.send(new PutObjectCommand({ Bucket: bucket, Key: key, Body: canonical, ServerSideEncryption: 'aws:kms', SSEKMSKeyId: kmsKeyId, ContentType: 'application/json', Metadata: { workflow_name: w.name, workflow_active: String(w.active) } }));`
      5. `await pg.query('INSERT INTO event_log (owner_id, kind, details, actor) VALUES ($1, $2, $3, $4)', ['kevin', 'n8n-workflows-archived', { workflow_id: w.id, workflow_name: w.name, sha256: sha, s3_key: key }, 'n8n-workflow-archiver-lambda']);`
      6. Return { workflow_id, sha256: sha, s3_key: key, size_bytes: canonical.length }.

    **`src/handler.ts`** (~60 lines):
    - Input: `{ workflows: N8nWorkflow[], bucket, prefix, kmsKeyId }`.
    - For each workflow, call `archiveWorkflow()`; collect results.
    - Return `{ archived: N, results: ArchiveResult[] }`.
    - Sentry wrap.

    **Flow:**

    **Stage 1 — Discover (safe, read-only):**
    1. Check env: `RDS_URL`, `AWS_REGION`. Fail if missing.
    2. SSH + `sudo systemctl status <n8n-unit-candidates>` — find the active unit name. Save to variable.
    3. SSH + `curl -s http://localhost:5678/rest/workflows | jq '.data | length'` — expect a number; fail if 404 (n8n already dead; abort).

    **Stage 2 — Archive (prerequisite for shutdown):**
    4. Open SSH local port-forward: `ssh -f -N -L 15678:localhost:5678 kevin@98.91.6.66`.
    5. `curl -s http://localhost:15678/rest/workflows > /tmp/n8n-workflows.json`.
    6. `curl -s http://localhost:15678/rest/workflows > /tmp/n8n-workflows-list.json` (for names+ids only).
    7. For each workflow ID in list, fetch full: `curl -s http://localhost:15678/rest/workflows/${id} > /tmp/n8n-workflow-${id}.json`.
    8. Close tunnel: `pkill -f 'ssh.*15678:localhost:5678' || true`.
    9. Invoke `services/n8n-workflow-archiver` Lambda via `aws lambda invoke` with the full workflow list as payload. Capture response; assert `archived == len(list)`.
    10. Print list of `s3_key`s archived.

    **Stage 3 — Shutdown (irreversible without restore from snapshot):**
    11. **Confirmation gate:** if NOT `--dry-run`, print summary of what's about to happen + prompt `read -p "Type 'decom' to proceed: " CONFIRM`. Exit if mismatch.

    13. SSH + `sudo systemctl disable <unit>`.
    14. SSH + `sudo systemctl mask <unit>`.
    15. Wait 10s; verify port locally-curl: `ssh kevin@98.91.6.66 'curl -s -o /dev/null -w %{http_code} http://localhost:5678' == 000 or timeout`.

    **Stage 4 — Firewall closure:**
    16. SSH + `sudo ufw deny 5678/tcp`.
    17. SSH + `sudo ufw reload`.
    18. SSH + `sudo ufw status numbered | grep 5678` — expect a DENY row.

    **Stage 5 — Audit trail:**
    19. `psql $RDS_URL -c "INSERT INTO event_log(owner_id, kind, details, actor) VALUES ('kevin', 'n8n-stopped', '{\"unit\":\"$unit\",\"host\":\"98.91.6.66\",\"workflows_archived\":$N,\"firewall\":\"ufw deny 5678/tcp\"}', 'decom-n8n.sh')"`.

    **Stage 6 — External probe:**
    20. Trigger `node scripts/verify-n8n-dead.mjs` (Task 3).

    **Dry-run:** Print all commands without executing Stage 3+4+5+6. Stage 1+2 always execute (read-only).

    **Cleanup on failure:** If Stage 2 (archival) fails, exit 1 — do NOT proceed to shutdown. The audit rule: never destroy without an archive.
